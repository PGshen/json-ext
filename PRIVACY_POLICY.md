# JSON-Ext 隐私政策

最后更新日期：2026-04-18

感谢您使用 JSON-Ext（以下简称“本扩展”）。本扩展是一个开源的浏览器扩展，用于查看、格式化和分析 JSON 数据。我们非常重视您的隐私，并尽可能以最小化方式处理数据。

## 1. 我们收集哪些数据

本扩展不会收集、存储或上传任何可识别个人身份的信息到开发者服务器。

本扩展仅在您的浏览器本地处理数据，并可能使用浏览器提供的本地存储能力（如 `chrome.storage`）保存以下配置项：

- 界面显示偏好（例如视图模式、展开状态）
- 格式化或展示相关选项
- 与功能体验相关的本地设置

这些信息仅存储在您的本地浏览器环境中，不会被我们主动传输到外部服务器。

## 2. 权限使用说明

本扩展可能申请以下权限，用于实现核心功能：

- `storage`：用于本地保存扩展设置与偏好。
- `tabs`：用于在用户主动使用扩展时识别当前标签页上下文。
- `host_permissions`（如 `<all_urls>`）：用于在不同站点页面中读取并处理 JSON 内容，以提供格式化、树形展示和调试能力。

我们不会将页面内容用于广告、画像或跨站跟踪。

## 3. 数据共享与第三方

- 我们不会出售您的数据。
- 我们不会将您的数据出租或交易给第三方。
- 我们不会出于广告定向目的共享您的数据。

## 4. 远程代码与外部请求

本扩展不依赖远程执行代码来实现核心功能。若未来版本引入网络请求或第三方服务，我们将同步更新本隐私政策并在发布信息中说明。

## 5. 数据安全

由于本扩展主要在本地运行，数据处理风险主要取决于您的浏览器环境与设备安全。建议您保持浏览器和操作系统为最新版本。

## 6. 开源说明

本扩展为开源项目，源代码可在 GitHub 仓库中查看。您可以审阅代码以了解数据处理逻辑与权限用途。

## 7. 政策变更

我们可能会根据功能更新、法律法规或平台要求对本政策进行调整。更新后将修改“最后更新日期”并在仓库中发布最新版本。
